Contributing to the PseudoDojo project

If you would like to contribute to the PseudoDojo project please visit the GitHub repository, where you should fork the project and commit your contributions accordingly.

If you contribute new functionality (e.g., a new test), please provide unit tests for your code. We cannot accept code additions without unittests.

Contributions of new pseudopotential sets should also be done via the GitHub repository. Where possible, perform the tests that are already present. Some tests, like the Delta gauge, require all-electron (AE) reference data that is only available for specific functionals; others do not.
